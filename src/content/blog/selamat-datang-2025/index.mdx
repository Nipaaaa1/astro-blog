Sekarang udah tahun 2025, dan di Dev Log kali ini, aku bakal sharing project pertamaku di tahun ini dan rencana project selanjutnya. Ya, karena aku udah lama gk bikin artikel baru, mungkin yg ini bakal singkat aja ya.

## Project pertama di tahun 2025
Sebenernya ini project yg udah lama mau aku bikin, cuman baru kesampaian sekarang. Project ini aku sebut _SYD Anime_ sebuah tempat streaming anime gratis yg aku bikin dari hasil scrape website _OtakuDesu_. Ya, ini bukan sebuah project yg bisa aku masukin ke portofolio, tapi setidaknya aku belajar cukup banyak soal SSR dan ngolah data dari API, yg mana cukup penting untuk project ku selanjutnya.

Aku publish project ini di grup fb IMPHNEN (Ingin Menjadi Programmer Handal namun Enggan Ngoding), karena udah banyak orang disana yg bikin project kayak ginian, ekspektasi ku cukup kecil soal project ini.

Untungnya, respon yg aku terima dari Project ini cukup positif. Ada yg muji kalau websitenya cepet, ada yg kasih masukan tentang UX dari websitenya, bahkan ada juga yg pengen tau gimana cara bikinnya. Aku belajar banyak dari project ini lah.

Kalau kamu pengen lihat project nya, [klik disini untuk lihat websitenya.](https://anime.syd.my.id/)

## Rencana project selanjutnya
Aku sebenernya agak dilema soal project selanjutnya, karena aku punya 3 ide berbeda yg masing - masing susah dibikin dan saling bertolak belakang. Ide pertama yaitu bikin sebuah website ecommerce yg konsepnya semacam _open source_ gitu, jadi aku bakal pake spesifikasi dari Open API untuk bikin dokumentasi API websitenya dan bikin semacam standar baru buat bikin toko online di Indonesia. Yep, it's sound like a daydream and too much to do with just an almost 8 year old phone, but I genuinely consider this.

Ide kedua adalah bikin sebuah SaaS tentang manajemen uang. Ini lebih _simpel_ dalam artian aku _cuman_ bikin sebuah platform untuk mempermudah orang catat pengeluaran / pemasukan mereka, ngeliat aktivitas keuangan mereka dan tentunya semuanya harus se-_painless_ mungkin. Jujur, ide ini ada di urutan terakhir dari 3 ide yg aku pikirin. Ini karena beda dengan 2 ide lainnya, aku cuman ngide buat bikin ini karena pengen cobain bikin SaaS aja. Perihal SaaS macam apa yg aku bikin itu gk terlalu penting, jadi untuk ide ini aku bakal simpen dulu.

Dan akhirnya ini lah yg aku pilih sebagai project selanjutnya, **Sosmed**.

Yep, you read that right, sosmed.

Aku gk berencana untuk bikin saingan facebook, tiktok atau x, aku cuman pengen cobain bikin sosmed sendiri. Sosmed ini terinspirasi dari Reddit dan X jadi hasilnya juga bakal kurang lebih mirip sama dua itu. Aku udah mulai cari referensi dan riset nya pas lagi ngetik ini. Targetnya sih, aku pengen dapet 100 user di sonmed ini, semoga aja kesampaian..... Semoga..... Oke, mungkin 100 user terlalu banyak jadi 50 aja lah.

Mungkin ini bakal makan waktu yg cukup lama jadi untuk artikel di blog ini kedepannya bakal jadi sedikit lebih di _tone down_. Walaupun begitu, aku tetep akan maintenance dan bikin artikel baru disini, cuman bakal lebih jarang dan _low effort_ aja, mungkin sekedar sharing progress dari project ini dan bahas teknologi baru yg menurutku menarik.

Sekian dulu untuk Dev Log kali ini, di artikel selanjutnya aku bakal ngelanjutin seri TypeScript kemarin yg belum selesai, see ya!
